iframe : html파일안에 html 파일을 포함시킨다.
	<iframe name="ff" src="myhome.html">
	
	iframe의 name의  값을 a 링크의 target 속성으로 지정한다.
	<a href ="아메리카노.html" target="ff">아메리카노 < /a >

span태그는 inline요소로 가로세로 크기가 조절되지 않는다
크기를 조절 하기 위해서 display : inline-block;스타일을 지정한다.
span{

	display: inline-block;
	width: 200px;    
	height:50px;
	margin-top: 30px;
}
