(입력은 prompt로 입력 받는다.-은 빼고 입력: 102131241) 1. A*2 + B*3 + ... + H*9 + I*2 + ... + L*5 의 총합을 구한다. 2. 1번의 합을 11로 나눈 나머지를 구한다. 3. 11에서 2번의 결과를 뺀다. 4. 3번의 결과가 0~9이면 값 그대로, 10이면 0, 11이면 1로 변환 5. 4번의 결과와 M자리의 값이 같으면 맞는 번호이다.
