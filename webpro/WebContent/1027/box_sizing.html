box-sizing :브라우저의 크기가 줄어들때 각 요소를 브러우저의 크기에 맞게
각 요소의 크기를 조절할때 필요하다

요소의 가로크기 width: calc(50% - 0.5%)
width : calc(300px - 1%)
width : calc(100px - 1px)


전체길이 300px 두개의 div가 각각 50%를 차지한다.
box-sizing: border-box 일때 border와 padding 속성을 포함해서  
가로크기를 50%로 한다.

box-sizing: border-box

50%
50%
전체길이 300px 두개의 div가 각각 50%를 차지한다
box-sizing :content-box일때 border와 padding 속성을 포함하지 않고
내용을 기준으로 50%를 차지하기 때문에 실제로 차지하는 크기는 50%가 넘는다.

box-sizing: border-box
